# 📧 AI Email Sender

Send professional emails instantly using AI 🚀

This app uses AI to generate and send well-structured emails for various purposes like job applications, referrals, leave requests, and more.

---

## ✨ Features

* 🤖 AI-generated professional emails
* 📩 Send emails directly from your Gmail
* 🔐 Secure (uses App Password, not your real password)
* 🎯 Supports multiple use cases:

  * Job application
  * Referral request
  * Leave request
  * General professional emails

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Agno Agent
* Google Gemini API

---

## 🚀 How to Run Locally

1. Clone the repo:
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name

2. Install dependencies:
   pip install -r requirements.txt

3. Create `.env` file:
   GOOGLE_API_KEY=your_api_key_here
   SENDER_EMAIL=[your_email@gmail.com](mailto:your_email@gmail.com)
   SENDER_PASSKEY=your_app_password

4. Run the app:
   streamlit run app.py

---

## 🔐 How Email Works

* Enter your Gmail address
* Enter Gmail App Password (not your real password)

👉 Emails will be sent from your own email securely.

---

## ⚠️ Important Note

* Do NOT use your real Gmail password
* Always use a Gmail App Password
* `.env` file is excluded for security
